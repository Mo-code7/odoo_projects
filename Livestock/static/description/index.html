Livestock Management System

🐾 إدارة الحيوانات

نظام متكامل لإدارة الحيوانات والماشية مع تتبع شامل لجميع البيانات المهمة.

  • تسجيل الحيوانات الجديدة مع بيانات شاملة
  • تتبع حالة الحيوان (حي، مريض، نافق، مباع)
  • إدارة الصور والملاحظات لكل حيوان
  • حساب العمر تلقائياً من تاريخ الميلاد
  • تصنيف الحيوانات حسب النوع والسلالة

✨ المميزات الرئيسية

  • واجهة عربية كاملة: جميع العناصر باللغة العربية مع دعم RTL
  • تصميم حديث: واجهة مستخدم أنيقة ومتجاوبة
  • إدارة الحالة: تتبع حالة الحيوان بسهولة
  • بحث متقدم: فلاتر شاملة للبحث والتصفية
  • عرض متعدد: قائمة، نموذج، و kanban
  • أمان كامل: صلاحيات محددة للمستخدمين

🚀 الاستخدام

  1. قم بتثبيت الموديول في Odoo
  2. اذهب إلى "إدارة المواشي" في القائمة الرئيسية
  3. اختر "الحيوانات" لبدء إدارة حيواناتك
  4. أضف حيوانات جديدة أو عدّل البيانات الموجودة
  5. استخدم الفلاتر للبحث والتصفية
